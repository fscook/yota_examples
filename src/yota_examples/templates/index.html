Yota

Yota is a Python based framework designed to make throwing together decent forms really fast, yet allowing flexibility for making complicated forms as well. The default library templates use Bootstrap and have support for AJAX based validation out of the box. Yota is small and well documented with the intent that if something isn't there, it's easy enough to patch it in. Coming in at under 500 lines of code complete comprehension of the libraries interworkings can be achieved quite rapidly.

Dynamic

    

    
Yota's internal Node and Validation systems allow for the addition of

    user to add an arbitrary number of dependents to a Form with JavaScript and

On-Submit AJAX

    

    
On-submit AJAX based validation allows submission and feedback of the
    Form without reloading the entire page. This is done in a lightweight way by
    parsing a JSON response, as opposed to re-rendering the entire Form and
    returning it.

Piecewise AJAX

        
    

    
Piecewise validation allows validation to be re-run based on JavaScript
    triggers. In the example below an AJAX request is made on the blur action of
    any input field. You may also notice that fields that haven't been modified
    yet will not recieve validation errors. This is because only form data for
    fields that have been triggered is submitted for validation.
