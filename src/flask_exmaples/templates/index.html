Yota


Flexible Python web form generation with asynchronous validation.



 Realtime Validation

Trigger a server side form validation with any JavaScript event on your input fields. Validation is carried out by reusable validation methods built into your schema.

 Sane Defaults

Default themed with Bootstrap with a wide array of default Nodes and validators makes mockups easy.

 Dynamic Forms

Dynamic form structures allow for complex forms with on the fly changes. Inject different input fields or validation methods into a specific instance of your Form where needed.

 Framework Agnostic

Designed to work well with Pyramid, Django, Flask, and likely many others.

 Hierarchical Schemas

Blueprints allow you to build Forms out of re-usable chunks that can be mix and matched, helping your forms be more DRY.

 Flexible Validation

Validation methods can accept an arbitrary array of Nodes that can be plugged in on a per-schema basis.
